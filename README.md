# tmQM-RDF Knwoledge Graph (v2025dev)

This is the read only archive of the tmQM-RDF knowledge graph (v2025dev).
For further details about the knowledge graph, see the [main repository](https://github.com/luca-cibinel/tmQM-RDF).

The script [interface.py](/interface.py) provides a convenient interface to the knowledge graph. It can be either imported into another python script or run directly to initialise the interface.
